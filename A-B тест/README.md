Описание проекта

А/В-тест

Мы — аналитик крупного интернет-магазина. Вместе с отделом маркетинга мы подготовили список гипотез для увеличения выручки.
Была проведена приоретизация гипотез, запуск A/B-теста и анализ результатов. На основании анализа теста было принято решение остановить тест и признать победу группы В.
